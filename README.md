# Market Elites Desktop

Desktop-focused trading dashboard with a dark/gold aesthetic.

**Live:** [elitesbeau.github.io/market-elites-desktop](https://elitesbeau.github.io/market-elites-desktop/)

## Features
- Wheel Tracker with full CSP/CC/assignment cycle management
- AI-powered trading signals with copy-trade workflow
- Signal performance analytics and discipline scoring
- Real-time presence tracking and admin tools
- TradingView chart integration and options chain viewer
- Expandable position rows with inline data
- Announcement ticker with Japanese typography design

## Built with
JavaScript, Firebase, TradingView, Tradier API, GitHub Pages
